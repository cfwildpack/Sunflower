# 🌻 Sunflower Project

A multi-language demo project inspired by sunflowers and nature.

## Features
- Sunflower growth simulation (Python)
- Garden controller (JavaScript)
- Nature web page (HTML/CSS)
- Simple API server (Node.js)
- Growth engine (C++)
- Plant model (Java)
- Sunlight calculator (Go)
- Diagnostics tool (Rust)
- Configuration files (JSON & YAML)
